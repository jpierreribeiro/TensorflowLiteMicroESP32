# Tensorflow lite no ESP32

Esse repositorio demonstra a utilização do tensorflow lite, aplicando conceitos de Machine Learning em sistemas embarcados ou de baixo poder computacional (Tiny ML) no ESP32.

O modelo foi treinado para fazer uma comparação entre dois números, com base nisso o código de inferência gera diferentes números e tenta descobrir qual o maior baseado nas probabilidades do output do modelo.
